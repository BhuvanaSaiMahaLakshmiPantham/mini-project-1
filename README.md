# Mini Project 1

College mini project with Frontend and Backend.

## Project Structure
- **Frontend/** - HTML page. Run using Live Server or open in Chrome.
- **backend/** - Node.js + Express server. Test APIs using Postman.

## How to Run Backend
1. Open terminal in `backend` folder
2. Run: `npm install`
3. Run: `node app.js`

## How to Run Frontend
Open `Frontend/index.html` file in Chrome or use VS Code Live Server.
